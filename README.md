# 硕士课题项目：基于AI（局域神经算子）的车辆空气动力学仿真

本项目是我硕士期间的核心研究课题，专注于应用AI（人工智能）解决复杂的计算流体动力学（CFD）问题。

本项目的核心目标是开发并实现一个**局域神经算子 (Local Neural Operator, LNO)**，使其能够作为一个**高效、可泛化的代理模型 (surrogate model)**，替代传统的有限元（FEM）求解器，用于实时预测**2D非定常流场**。

这项研究的成果可直接应用于【载运工具运用工程】领域，为车辆的空气动力学设计、优化和流场分析提供超高效的计算工具。

## 🔬 方法论：为何LNO可以实现泛化？

传统的AI模型（如FNO）在固定的几何域上学习，导致其无法泛化。本项目的LNO模型通过两项关键创新克服了这一难题：

1.  [cite_start]**学习物理算子:** LNO不学习特定形状的“解”，而是学习物理控制方程（N-S方程）的**局部、位移不变的时间推进算子 (GLG_L)** [cite: 3649, 3688, 3693]。这成功地将“物理规律”与“特定几何”解耦。

2.  [cite_start]**混合双路径架构:** LNO的内部块**并行融合了“物理路径”和“谱路径”** [cite: 378-379]。
    * [cite_start]**物理路径 (Physical Path):** 作为一个**局部CNN（卷积）**，它在物理网格上运行，负责**捕捉高频局部细节**，并**修正谱路径在尖锐梯度处（如边界层）产生的数值振荡** [cite: 1176-1177, 1180-1182]。
    * [cite_start]**谱路径 (Spectral Path):** 作为**主导路径**，它通过**勒让德谱变换**在模态空间中运行，负责**高效捕捉流场的全局低频趋势** [cite: 1176-1177]。

通过这种方式，LNO模型实现了**100倍至1000倍**于传统FEM求解器的计算加速，同时准确捕捉了关键的流场物理现象。
